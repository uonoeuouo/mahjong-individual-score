# Discord 麻雀スコア自動集計Bot

Discordのチャットに投稿された麻雀のスコアを読み取り、Googleスプレッドシートに自動で集計・保存するBotです。
増分更新（前回の続きから読み込み）、名前の表記揺れ吸収、スタッツの自動計算に対応しています。

## 💬 ユーザーサポート Q&A

### Q1. スコアの投稿フォーマットは？

**A.** 以下の形式で投稿してください（名前と点数の間はスペース）:
```
A さん 54.3
B さん 12.1
C さん -15.5
D さん -50.9
```

**重要:** 4 人の合計は必ず「0」になる必要があります。

---

### Q2. チョンボが発生したときは？

**A.** スコアの末尾に「チョンボ」と入力してください:
```
Aさん 54.3
Bさん 12.1 チョンボ
Cさん -15.5
Dさん -50.9
```

複数回の場合は「チョンボ2」のように数字を付けます:
```
C さん -15.5 チョンボ2
```

---

### Q3. 投稿したスコアを修正したい

**A.** 以下の手順で対応してください:

1. **修正前に:** Discord の該当投稿を削除
2. **新しい投稿:** 正しいスコアを投稿
3. **集計実行:** `!collect` コマンドで再集計
4. ※ 過去分を修正する場合は、管理者に連絡してください

---

### Q4. 同点の場合の順位は？

**A.** 同点時は平均順位で計算されます:

例:
- 得点 70, -5, -5, -60 の場合
  - 順位: 1 位, 2.5 位, 2.5 位, 4 位

スプレッドシートの「着数」欄で小数値が表示される場合があります。これは正常な動作です。

---

### Q5. 集計後にスプレッドシートの内容が違う

**A.** 以下を確認してください:

1. ✅ 投稿した時刻が集計時刻より前か確認
2. ✅ 名前の表記揺れ（例：「太郎」→「たろう」）がないか確認
3. ✅ Internet 接続が正常か確認
4. それでも解決しない場合は管理者に連絡してください
